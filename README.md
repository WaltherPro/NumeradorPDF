# FolioMark — Numerador de PDF Online

Herramienta gratuita para numerar páginas de documentos PDF con estilos elegantes.

## Funciones
- Numeración normal (1 → N) e inversa (N → 1)
- Estilos: arábigo, romano, letras, círculo, formal, guiones
- Vista previa en tiempo real
- Posición libre (6 zonas de la página)
- Color personalizable
- Sin registro, sin límites

## Tecnologías
- HTML5 / CSS3 / JavaScript puro
- [pdf-lib](https://pdf-lib.js.org/) para edición de PDF
- [PDF.js](https://mozilla.github.io/pdf.js/) para vista previa
